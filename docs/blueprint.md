# **App Name**: H-TV: Streaming Hub

## Core Features:

- User Authentication: Secure user authentication using Firebase (email/password, Google Sign-in).
- Content Browsing: Browse movies and TV shows fetched from TMDB with filtering and pagination.
- Video Streaming: Stream videos with a custom video player supporting multiple quality levels and playback controls.
- Search Functionality: Implement a search feature to find movies and TV shows using the TMDB API.
- Comment System: Enable users to post comments and nested replies using Firebase.
- Profile Customization: Allow users to create a profile and customize the content by liking/disliking it.
- AI-Powered Recommendation Tool: Use user's viewing history and like/dislike data in Firebase database to offer viewing suggestions and recommendations

## Style Guidelines:

- Primary color: Deep midnight blue (#0A192F) to establish a sleek and immersive viewing atmosphere.
- Background color: Dark slate gray (#172A45), desaturated for minimal distraction and a comfortable viewing experience.
- Accent color: Electric purple (#BD93F9), placed carefully for highlights on interactive elements, and calls to action, to guide the user.
- Body and headline font: 'Inter', sans-serif, provides a modern and neutral aesthetic for comfortable readability.
- Use minimalist and clear icons for navigation and video controls.
- Implement a responsive, mobile-first layout using CSS Grid/Flexbox.
- Employ subtle CSS animations for loading states, transitions, and enhanced UX.
